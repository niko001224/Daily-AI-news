## 📌 项目简介

本仓库用于**自动化爬取、存储与整理 AI 行业资讯及国内云厂商动态**，所有内容以 **Markdown** 格式进行结构化沉淀，构建一个长期可积累、可分析、可回溯的行业资讯知识库。

**系统机制说明：**
- ⏰ 每日 **09:00** 定时任务
- 🔍 自动爬取 **过去 24 小时内** 的最新资讯
- 📥 覆盖内容包括：
  - AI 行业新闻、技术进展、模型发布、产品动态
  - 国内云厂商相关动态（云服务、算力平台、AI 平台、行业解决方案等）

## 📁 目录结构说明

- **每个文件夹对应一周的资讯数据**
- 文件夹内按来源与主题存放当周爬取的 Markdown 文档：
  - AI 行业资讯
  - 国内云厂商资讯
- 按时间维度归档，支持历史回溯、趋势分析与结构化整理

## 📊 周报生成机制

- 每周自动读取对应时间文件夹内的所有 Markdown 文档
- 对当周 AI 行业动态与云厂商资讯进行整理、聚合与归类
- 生成一份 **周度综合资讯报告**
- 统一存储至报告文档中，作为阶段性总结与趋势参考

## 🎯 项目目标

- 构建一个**自动更新的 AI + 云计算行业情报系统**
- 实现 **资讯获取 → 结构化存储 → 周期性总结** 的闭环
- 为行业研究、技术调研、产品分析与战略判断提供数据基础支撑
